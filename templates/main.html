API Basejump: URL Shortener Microservice

    User Stories

      1) If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.

      2) When I visit that shortened URL, it will redirect me to my original link.

    Example usage:

    https://shorturl.herokuapp.com/new/http://google.com
    https://shorturl.herokuapp.com/new/http://foo.bar

    Example output:

    { "URL": http://google.com, "shor_URL": "efgh" }

    Then, visit https://shorturl.herokuapp.com/"your short_URL"
